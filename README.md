# First React Landing Page

My first React landing page built with custom CSS and Vite. A simple, responsive single-page application showcasing foundational React concepts.

## Tech Stack

- **Framework**: React
- **Language**: JavaScript
- **Build Tool**: Vite
- **Styling**: Custom CSS

## Getting Started

1. Clone the repository
2. Install dependencies: \
pm install\
3. Run development server: \
pm run dev\
